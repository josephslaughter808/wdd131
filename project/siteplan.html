Pucklytics Site Plan

A mobile-first adult hockey app focused on stats, scheduling, highlights, draft tools, film review, and league communication.

Site Name

Pucklytics

This name combines puck with analytics, which fits the app's purpose as a hockey platform built around player stats, game film, highlights, and league management.

Optional domain idea: pucklytics.app

Site Purpose

Pucklytics provides adult hockey leagues with a central mobile-friendly hub where players, captains, and commissioners can manage schedules, view stats, upload and review film, track highlights, communicate through team and league feeds, and run live drafts.

Scenarios

  • When is my next game, what rink is it on, and have the lines been posted yet?
  • How do I compare to the rest of my team and league in goals, assists, and plus/minus?
  • Can I watch a tagged clip of a player's goal before drafting them?
  • How can a commissioner quickly see who is signed up for the league and assign captains or subs?

Color Schema

Deep Navy — #07111F

Used for page backgrounds, cards, and the overall dark app shell.

Glow Blue — #38BDF8

Used for outlines, active states, highlights, and important focus elements.

Burnt Orange — #F97316

Used for buttons, calls to action, and competitive game-day accents.

This site plan document uses the selected color palette so the plan reflects the real brand direction.

Typography

Oswald

Used for main headings, section titles, and bold hockey-style emphasis.

Source Sans 3

Used for body text, descriptions, labels, and readable app content.

Development Notes

The final project will continue to evolve, but the first priority is a polished mobile-first experience. The main views planned for the finished project are Home, Stats, Feed, Film, Draft, Chat, Profile, and Commissioner tools.
